A maior parte das redes sociais utiliza o modelo cliente-servidor.

Neste modelo, o utilizador acede à rede social através de uma página Web ou aplicação móvel.

A interação com a interface do utilizador resulta de pedidos feitos ao servidor que contêm, na maioria das redes sociais, uma Application Programming Interface (API) com vários micro serviços.

A utilização deste tipo de arquitetura permite escalabilidade e uma fácil manutenção, uma vez que, a API está divida em vários micro serviços e se um deixar de funcionar, apenas esta funcionalidade é afetada
