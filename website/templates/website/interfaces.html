Interfaces

As interfaces de utilizador das redes sociais são um factor determinante para o seu sucesso.

Idealmente, estas têm de ser fáceis de usar e possuir um design atrativo. O seu desenho é feito de modo a captar a maior atenção do utilizador possível.

A maior parte das redes sociais utiliza uma lista cronológica de publicações, ou seja, estas são apresentadas pela ordem em que foram publicadas. A navegação é feita através de um gesto para cima ou para baixo conforme se queira ir para as publicações mais antigas ou para as mais recentes, respetivamente.
